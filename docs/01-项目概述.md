# 项目概述

## 1.1 项目背景

### 市场需求

在现代前端开发中，设计系统（Design System）和组件库的建设变得越来越重要。Storybook 作为业界标准的 UI 组件开发工作平台，被 Shopify、Airbnb、Stripe 等众多知名企业采用。然而，从设计稿到 Storybook 组件的转换过程往往是手动、重复且容易出错的工作。

### 现有方案分析

| 工具 | 方向 | Stars | 局限性 |
|------|------|-------|--------|
| storybook-to-figma | Storybook → Figma | 19 | 单向同步，不支持 Figma → Storybook |
| storysync | Storybook ↔ Figma 同步 | 15 | 仅限简单组件，缺乏深度转换 |
| figmator | Figma → React + Storybook | 12 | 项目已不活跃，仅支持基础组件 |
| design-system-hub | 零配置 Storybook + Figma 同步 | 17 | 是替代品，非转换器 |

### 市场空白

1. **没有真正可靠的 Figma → Storybook 转换器**
2. 现有方案停留在"同步"而非"智能转换"
3. 缺乏对复杂组件（Variant、Component Properties）的支持
4. 缺少完整的设计 Token 提取

## 1.2 项目定位

### 核心价值

> **"One-click from Figma design to production-ready Storybook components"**

一键将 Figma 设计转换为生产就绪的 Storybook 组件。

### 目标用户

1. **Design System 团队**
   - 快速将设计稿转换为开发组件
   - 保持设计与代码的一致性
   - 减少设计-开发沟通成本

2. **前端开发者**
   - 减少重复性组件开发工作
   - 加速组件库建设
   - 获取准确的类型定义

3. **全栈设计师**
   - 独立完成设计到开发的闭环
   - 快速验证设计可行性
   - 降低开发门槛

## 1.3 功能范围

### 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| Figma 设计提取 | 从 Figma Plugin 导出结构化 JSON | P0 |
| React 组件生成 | 生成标准的 React 组件代码 | P0 |
| Storybook Story 生成 | 生成带 Controls 的 Story 文件 | P0 |
| CSS 样式导出 | 提取并转换设计样式 | P0 |
| Variant 支持 | 处理 Figma Variant 组件 | P1 |
| TypeScript 类型推断 | 自动推断 Props 类型 | P1 |
| Vue 3 支持 | Vue 组件生成 | P2 |
| Angular 支持 | Angular 组件生成 | P2 |
| 设计 Token 提取 | 提取颜色、字体、间距等 Token | P1 |
| MDX 文档生成 | 自动生成组件文档 | P2 |

### 不在范围内

- Figma 账号登录/认证
- 云端存储服务
- 组件发布平台
- 设计审查/协作功能

## 1.4 成功指标

### 量化指标

| 指标 | 目标值 |
|------|--------|
| 基础组件转换成功率 | ≥ 90% |
| 复杂组件（Variant）转换成功率 | ≥ 70% |
| 代码可读性评分 | ≥ 8/10 |
| 生成代码可直接运行率 | ≥ 85% |

### 质量指标

- 生成的 TypeScript 代码无类型错误
- Storybook Story 可正常渲染
- CSS 样式与设计稿一致度 ≥ 95%

## 1.5 技术选型背景

### 技术趋势

1. **AI-First Development**
   - Storybook v10.3.5 已开始集成 MCP
   - AI 生成组件、Story、文档是强需求

2. **Modern Build Tools**
   - Vite-first approach
   - ESM-native

3. **Testing Evolution**
   - Vitest 替代 Jest
   - Playwright 集成

### 为什么选择这些技术

| 技术 | 选择理由 |
|------|----------|
| TypeScript | 类型安全、Figma Plugin 官方支持 |
| React | Storybook 官方首选、用户基数大 |
| Node.js | 跨平台、成熟的工具链 |
| esbuild | 极速打包、开发体验好 |
| Vitest | 快速、兼容 Jest API |
